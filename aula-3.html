Atributos em HTML

Um atributo em HTML é uma informação extra que você adiciona a uma tag para modificar seu comportamento ou fornecer mais detalhes sobre aquele elemento.

Como Funciona?

  • Os atributos sempre ficam na tag de abertura
  • Eles tem a estrutura nome="valor"
  • Permitem personalizar ou especificar algo que a tag vai fazer, sem precisar mudar o conteúdo.

Exemplo:


<p title="Parágrafo de introdução"> 
	Hoje começamos a aprender sobre atributos em HTML!
</p>

Saida:

Hoje começamos a aprender sobre atributos em HTML!

Exemplo 2:


<p style="color: red;"> 
	O Texto deste parágrafo é vermelho!
</p>

Saida:

O Texto deste parágrafo é vermelho!

  • id – Identifica um elemento único na página.
  • class – Agrupa elementos para estilização ou manipulação via CSS/JS.
  • href – URL de destino de um link (<a>).
  • src – Caminho de uma imagem,mídia ou script js (<img>, <script>).
  • type – Especifica o tipo de entrada de um formulário (<input> ou <button>).
  • placeholder – Texto de sugestão dentro de campos de formulário (<input>).

Atributo class

O atributo class permite atribuir um ou mais grupos de estilo a um elemento, que podem ser usados pelo CSS ou manipulados via JavaScript.

  • Permite estilizar múltiplos elementos de forma igual
  • Facilita manipulação via JavaScript

Exemplo


<!-- HTML -->
<ol>
	<li class="destaque">O parágrafo terá o texto azul.</li>
	<li class="importante">O texto será em negrito.</li>
	<li class="destaque importante">Este estará em negrito e azul.</li>	
</ol>


#CSS	
.destaque { 
	color: blue; 
}
.importante { 
	font-weight: bold; 
}

Saida:

  1. O parágrafo terá o texto azul.
  2. O texto será em negrito.
  3. Este estará em negrito e azul.

O Atributo id

O atributo id identifica um único elemento na página. É muito útil para CSS e JavaScript.

  • Deve ser único – não pode repetir na mesma página
  • Usado para CSS – estilizar um elemento específico
  • Usado para JavaScript – manipular elementos diretamente
  • Referência em links – criar âncoras internas na página

Exemplo - Manipular directamente um elemento via Js


<!-- HTML -->
<h1 id="titulo-principal">Bem-vindo ao meu site!</h1>
						

// JavaScript
const idH1 = document.getElementById("titulo-principal");

idH1.onclick = () => {
	idH1.style.color = "red";
	idH1.style.backgroundColor = "yellow";
	idH1.textContent = "Você clicou no título!";
}

Saida:

Bem-vindo ao meu site!

Diferença entre id e class

  • ID – Identifica um elemento único na página.
  • CLASS – Pode agrupar vários elementos com o mesmo estilo ou função.
  • Ambos podem ser usados em CSS e JavaScript. 

<ul>
	<li class="paragrafo-grupo" > Paragrafo sem estilo especial.</li>
	<li class="paragrafo-grupo" > Paragrafo sem estilo especial.</li>
	<li class="paragrafo-grupo" > Paragrafo sem estilo especial.</li>
	<li class="paragrafo-grupo" > Paragrafo sem estilo especial.</li>	
</ul>
<button id="botao">Clique-me</button>
						

const botao = document.getElementById("botao");
botao.onclick = alterarParagrafos;

function alterarParagrafos(){
	const grupos = document.getElementsByClassName("paragrafo-grupo");
	Array.from(grupos).forEach((grupo, indice) => {
		grupo.style.color = "blue";
		grupo.style.fontWeight = "bold";
		grupo.textContent = `Você alterou o parágrafo ${indice + 1}º do grupo!`;
	});
}

Saida

  • Paragafo sem estilo especial.
  • Paragafo sem estilo especial.

Links em HTML

O elemento <a> cria links para navegar em outras páginas, seções ou arquivos.

Atributos do link

  • href – Define o destino do link (URL ou âncora interna).
  • target – Define onde o link será aberto (_blank para nova aba).

Exemplo - Link que irá abrir para a página do minecraft


<a href="https://www.minecraft.net/pt-br/" >
	Visite o site do Minecraft!
</a>

Saida:

Visite o site do Minecraft!

Exemplo 2 - Link que irá abrir nova aba para a página do minecraft


<a href="https://www.minecraft.net/pt-br/" target="_blank" >
	Visite o site do Minecraft em nova Aba!
</a>

Saida:

Visite o site do Minecraft em nova Aba!

Exemplo 3 - Link que irá abrir um documento


<a href="./exemplos/aula-3/Demo-aula3-PW.pdf">
	Visualizar documento PDF
</a>

Saida:

Visualizar documento PDF

Exemplo 3 - Link que irá baixar um documento


<a href="./exemplos/aula-3/Demo-aula3-PW.pdf" download >
	Baixar documento PDF
</a>

Saida:

Baixar documento PDF

NOTE: para baixar o documento foi necessário usar o atributo download

Âncoras Internas

Você pode criar links que levam a uma seção específica da mesma página usando #id:

  • Crie um elemento com id único.
  • Use <a href="#id"> para navegar até ele.

Exemplo - Âncoras Internas


						<h1>Página com Âncoras</h1>

<!-- Menu de Navegação -->
<nav id="topo" >
  <a href="#secao1">Seção 1</a> | 
  <a href="#secao2">Seção 2</a> | 
  <a href="#secao3">Seção 3</a> | 
  <a href="#topo">Voltar ao Topo</a>
</nav>

<!-- Seções -->
<section id="secao1">
  <h2>Seção 1</h2>
  <p>Este é o conteúdo da seção 1. Aqui mostramos como usar uma âncora para acessar diretamente esta parte da página.</p>
</section>

<section id="secao2">
  <h2>Seção 2</h2>
  <p>Este é o conteúdo da seção 2. Repare que, ao clicar no link correspondente no menu, você é trazido diretamente para cá.</p>
</section>

<section id="secao3">
  <h2>Seção 3</h2>
  <p>Este é o conteúdo da seção 3. As âncoras facilitam a navegação em páginas mais longas.</p>
</section>

<!-- Âncora para o topo -->
<a href="#topo"><Voltar ao topo/a>

Imagens em HTML

A tag <img> é usada para inserir imagens em uma página.

Ela é uma tag vazia, ou seja, não possui conteúdo dentro dela.

Atributos da <img>

  • src: caminho da imagem (local ou URL)
  • alt: texto alternativo (essencial para acessibilidade e SEO)
  • width e height: definem tamanho da imagem

Exemplo de Imagem


<img src="exemplos/aula-3/minecraft.jpeg"
     alt="Foto de Minecraft"
     width="300" 
     height="200">
